# 内控矩阵 接口文档

> Base URL: `/api`  
> 统一响应格式：`{ code: 0, msg: "success", data: ... }`  
> `code !== 0` 时视为业务错误，`msg` 为错误描述。

---

## 1. 获取流程分组列表

用于内控矩阵首页展示字母索引 + 流程列表。

**请求**

```
GET /api/matrix/process/list
```

**响应示例**

```json
{
  "code": 0,
  "msg": "success",
  "data": {
    "groups": [
      {
        "letter": "A",
        "items": [
          { "name": "按揭贷款管理", "code": "A001" },
          { "name": "安全生产管理", "code": "A002" }
        ]
      },
      {
        "letter": "B",
        "items": [
          { "name": "保全服务流程管理", "code": "B001" },
          { "name": "保险核保管理",     "code": "B002" }
        ]
      }
    ]
  }
}
```

**字段说明**

| 字段 | 类型 | 说明 |
|------|------|------|
| `groups` | Array | 按字母分组的流程列表 |
| `groups[].letter` | String | 分组字母（大写） |
| `groups[].items` | Array | 该字母下的流程项 |
| `groups[].items[].name` | String | 流程名称 |
| `groups[].items[].code` | String | 流程编码（唯一） |

## 2. 获取控制流程风险列表（含动态列配置）

用于控制流程详情页，展示指定流程的风险点表格。  
后端返回完整列定义，前端取前 3 列作为表格列，其余列在点击展开后的详情区展示。

**请求**

```
GET /api/matrix/process/:code/risks
```

**路径参数**

| 参数 | 类型 | 说明 |
|------|------|------|
| `code` | String | 流程编码，如 `B001` |

**响应示例**

```json
{
  "code": 0,
  "msg": "success",
  "data": {
    "processName": "保全服务流程管理",
    "columns": [
      { "key": "riskName",        "label": "风险点名称",       "flex": 2 },
      { "key": "controlMeasure",  "label": "控制措施描述",     "flex": 3 },
      { "key": "controlLevel",    "label": "执行控制层级",     "flex": 2 },
      { "key": "headDeptHQ",      "label": "总部牵头部门",     "flex": 2 },
      { "key": "headDeptProvince","label": "省公司牵头部门",   "flex": 2 },
      { "key": "headDeptCity",    "label": "地市公司牵头部门", "flex": 2 },
      { "key": "headDeptCounty",  "label": "县支公司牵头部门", "flex": 2 },
      { "key": "controlSystem",   "label": "控制制度名称",     "flex": 2 },
      { "key": "controlDesc",     "label": "控制制度描述",     "flex": 3 }
    ],
    "riskItems": [
      {
        "riskName": "精算评估方法不符合会计准则",
        "controlMeasure": "精算部和财务部跟踪相关监管机构发布的会计准则等",
        "controlLevel": "总公司",
        "headDeptHQ": "精算部",
        "headDeptProvince": "无",
        "headDeptCity": "无",
        "headDeptCounty": "无",
        "controlSystem": "日常步骤操作",
        "controlDesc": "不适用"
      }
    ]
  }
}
```

**字段说明**

| 字段 | 类型 | 说明 |
|------|------|------|
| `processName` | String | 流程名称 |
| `columns` | Array | 列定义数组，顺序即展示顺序，由后端控制 |
| `columns[].key` | String | 字段名，与 `riskItems` 对象的属性名对应 |
| `columns[].label` | String | 列标题文字 |
| `columns[].flex` | Number | 列宽比例（对应 CSS grid `fr` 单位） |
| `riskItems` | Array | 风险点列表，每项为扁平对象，字段名与 `columns[].key` 一一对应 |

**前后端职责说明**

| 职责 | 后端 | 前端 |
|------|------|------|
| 列定义（名称、顺序、宽度） | ✅ 通过 `columns` 返回 | 只读取，不硬编码 |
| 表格列（前 3 列） | 不感知 | 取 `columns.slice(0, 3)` 渲染 |
| 详情列（第 4 列起） | 不感知 | 取 `columns.slice(3)` 渲染展开区 |
| 行数据 | ✅ 扁平结构，含所有字段 | 按 `key` 取值，动态渲染 |

## 错误码

| code | 说明 |
|------|------|
| 0 | 成功 |
| 1001 | 流程编码不存在 |
| 5000 | 服务器内部错误 |
