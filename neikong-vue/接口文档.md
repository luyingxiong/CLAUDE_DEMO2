# 内控矩阵 接口文档

> Base URL: `/api`  
> 统一响应格式：`{ code: 0, msg: "success", data: ... }`  
> `code !== 0` 时视为业务错误，`msg` 为错误描述。

---

## 1. 获取流程分组列表

用于内控矩阵首页展示字母索引 + 流程列表。

**请求**

```
GET /api/matrix/process/list
```

**响应示例**

```json
{
  "code": 0,
  "msg": "success",
  "data": {
    "groups": [
      {
        "letter": "A",
        "items": [
          { "name": "按揭贷款管理", "code": "A001" },
          { "name": "安全生产管理", "code": "A002" }
        ]
      },
      {
        "letter": "B",
        "items": [
          { "name": "保全服务流程管理", "code": "B001" },
          { "name": "保险核保管理",     "code": "B002" }
        ]
      }
    ]
  }
}
```

**字段说明**

| 字段 | 类型 | 说明 |
|------|------|------|
| `groups` | Array | 按字母分组的流程列表 |
| `groups[].letter` | String | 分组字母（大写） |
| `groups[].items` | Array | 该字母下的流程项 |
| `groups[].items[].name` | String | 流程名称 |
| `groups[].items[].code` | String | 流程编码（唯一） |

## 2. 获取控制流程风险列表

用于控制流程详情页，展示指定流程的风险点 + 控制措施表格。

**请求**

```
GET /api/matrix/process/:code/risks
```

**路径参数**

| 参数 | 类型 | 说明 |
|------|------|------|
| `code` | String | 流程编码，如 `B001` |

**响应示例**

```json
{
  "code": 0,
  "msg": "success",
  "data": {
    "processName": "保全服务流程管理",
    "riskItems": [
      {
        "id": 1,
        "riskName": "精算评估方法不符合会计准则",
        "controlMeasure": "精算部和财务部跟踪相关监管机构发布的会计准则等",
        "controlLevel": "总公司",
        "details": {
          "riskName": "精算评估方法不符合会计准则",
          "controlMeasure": "精算部和财务部跟踪相关监管机构发布的会计准则等",
          "controlLevel": "总公司",
          "headDeptHQ": "精算部",
          "headDeptProvince": "无",
          "headDeptCity": "无",
          "headDeptCounty": "无",
          "controlSystem": "日常步骤操作",
          "controlDesc": "不适用"
        }
      }
    ]
  }
}
```

**字段说明**

| 字段 | 类型 | 说明 |
|------|------|------|
| `processName` | String | 流程名称 |
| `riskItems` | Array | 风险点列表 |
| `riskItems[].id` | Number | 风险点 ID |
| `riskItems[].riskName` | String | 风险点名称 |
| `riskItems[].controlMeasure` | String | 控制措施描述 |
| `riskItems[].controlLevel` | String | 执行控制层级 |
| `riskItems[].details` | Object\|null | 展开后的详细信息，无则为 `null` |
| `details.headDeptHQ` | String | 总部牵头部门 |
| `details.headDeptProvince` | String | 省公司牵头部门 |
| `details.headDeptCity` | String | 地市公司牵头部门 |
| `details.headDeptCounty` | String | 县支公司牵头部门 |
| `details.controlSystem` | String | 控制制度名称 |
| `details.controlDesc` | String | 控制制度描述 |

---

## 错误码

| code | 说明 |
|------|------|
| 0 | 成功 |
| 1001 | 流程编码不存在 |
| 5000 | 服务器内部错误 |
